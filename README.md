# slopground

Experimental playground for creative coding projects and interactive tools.

## Projects

### webcamHandTracker

Real-time hand tracking using Google's MediaPipe library for gesture-based code control.

**Features:**
- Tracks 21 hand landmarks (fingers, wrist, palm, forearm)
- Real-time gesture recognition
- Built-in calibration routine
- Interactive visualization
- Designed for gestural control of code and applications

**Quick Start:**
```bash
cd webcamHandTracker
npm install
npm start
```

See [webcamHandTracker/README.md](webcamHandTracker/README.md) for detailed documentation.
